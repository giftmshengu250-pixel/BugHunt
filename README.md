# Bug Hunt Project

## Overview

This project is a collaborative debugging exercise completed by a team of five developers. The goal was to identify bugs in a JavaScript application, assign one bug to each team member, fix the issues individually, and then merge all changes into a single working version using Git and GitHub.

## Objectives

* Identify bugs in the JavaScript code.
* Assign one bug to each team member.
* Fix bugs independently.
* Practice version control using Git.
* Collaborate through GitHub by merging changes into a shared project.

## Bugs Fixed

1. Changed the Add button event listener from `mouseover` to `click`.
2. Corrected the task ID comparison in the `toggleTask()` function.
3. Fixed the Active filter to display incomplete tasks.
4. Corrected the Delete button functionality to pass the correct task ID.
5. Assisted with testing, reviewing changes, and validating the final merged application.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## Team Workflow

1. Each team member was assigned a specific bug.
2. Individual fixes were implemented and tested locally.
3. Changes were committed using Git.
4. Team members collaborated through GitHub.
5. All fixes were merged into the final project version.

## Outcome

The project was successfully debugged and merged, resulting in a fully functional application while providing hands-on experience with collaborative development and version control workflows.
